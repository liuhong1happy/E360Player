Quick start

  1. clone & run

    git clone git@github.com:liuhong1happy/E360Player.git
cd E360Player
npm install && npm start

  2. release

    # osx 
npm run pack-osx
# windows 32
npm run pack-win32
# windows 64
npm run pack-win64
# linux
npm run pack-linux
